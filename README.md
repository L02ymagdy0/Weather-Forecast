# 🌦️ Dynamic Weather Forecast Application

A sleek, modern, and fully responsive web application that provides real-time weather updates and a multi-day forecast. Built using asynchronous JavaScript, WeatherAPI, Tailwind CSS, and Font Awesome.

## 🚀 Features
* **Real-time Search:** Instantly fetch weather data for any city worldwide.
* **Smart Reset:** Automatically resets and defaults back to Cairo the exact moment the search input is cleared.
* **Utility-First Responsive Design:** Styled entirely with **Tailwind CSS** for a fluid, modern interface that looks great on mobile, tablet, and desktop.
* **Visual Telemetry:** Uses **Font Awesome** icons to dynamically represent wind, humidity, and weather conditions for an intuitive user experience.
* **3-Day Forecast:** Shows up-to-date conditions, minimum, and maximum temperatures for upcoming days.
* **Asynchronous Data:** Built using `async/await` fetch requests for seamless performance without page reloads.

## 🛠️ Tech Stack
* **Frontend:** HTML5, Tailwind CSS, JavaScript (ES6+)
* **Icons:** Font Awesome
* **API:** [WeatherAPI](https://www.weatherapi.com/)

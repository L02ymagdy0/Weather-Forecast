🌦️ Dynamic Weather Forecast Application

A sleek, responsive web application that provides real-time weather updates and a multi-day forecast using asynchronous JavaScript and the WeatherAPI.

## 🚀 Features
* **Real-time Search:** Instantly fetch weather data for any city worldwide.
* **Smart Reset:** Automatically resets and defaults back to Cairo if the search input is cleared.
* **Current Weather Metrics:** Displays temperature, humidity, wind speed, and wind direction.
* **3-Day Forecast:** Shows up-to-date conditions, minimum, and maximum temperatures for the upcoming days.
* **Asynchronous Data:** Built using `async/await` fetch requests for seamless performance without page reloads.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **API:** [WeatherAPI](https://www.weatherapi.com/)
